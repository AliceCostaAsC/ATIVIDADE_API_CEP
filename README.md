#Como Funciona

O Projeto criado através da API feita pelo site https://viacep.com.br/, que ao inserir o CEP de uma cidade o programa traz vários dados,tais como logradouro, UF, GIA...

# Referência

https://viacep.com.br/

# Tecnologias Usadas

Html5, Css3, JavaScript e Bootstrap
